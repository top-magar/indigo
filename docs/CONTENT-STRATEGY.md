# Indigo — Content Marketing & SEO Strategy

## SEO Issues Found (24 items)

### Critical
- Blog pages missing OG images, canonical URLs, twitter cards
- Checkout/wishlist/search pages not set to `noindex`
- robots.ts missing `/admin/`, `/editor/`, `/auth/` disallow rules
- Sitemap missing category pages and blog posts
- Store FAQ page not using FAQJsonLd (component exists but unused)

### Medium
- Root layout missing og:image in openGraph config
- Root layout locale should be `en_NP` not `en_US`
- Marketing layout has no metadata
- Store product listing missing ItemListJsonLd
- Store about/contact/FAQ pages have minimal metadata
