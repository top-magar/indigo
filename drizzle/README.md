# Drizzle Migrations

This folder contains auto-generated SQL migrations from Drizzle ORM.

## Important Notes

1. **DO NOT manually edit files in this folder** - They are auto-generated by `drizzle-kit generate`
2. **DO NOT add manual SQL files here** - Use `scripts/supabase/` for manual migrations
3. The `meta/` folder contains migration metadata - do not modify

## Current Migrations

| File | Description |
|------|-------------|
| `0000_silky_ultimates.sql` | Initial schema (tenants, users, products) |
| `0001_lonely_proteus.sql` | Orders, variants, inventory levels |
| `0002_order_inventory_enhancements.sql` | Order and inventory enhancements |
| `0003_glossy_sugar_man.sql` | Categories, domains, product fields |
| `0004_noisy_wolfpack.sql` | Media library tables |

## Usage

For local development with Drizzle:
```bash
# Generate migrations from schema changes
pnpm drizzle-kit generate

# Push migrations to database
pnpm drizzle-kit push

# Open Drizzle Studio
pnpm drizzle-kit studio
```

## For Supabase

Use the scripts in `scripts/supabase/` folder instead. Those are designed to be run in the Supabase SQL Editor.
