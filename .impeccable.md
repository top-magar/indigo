# Indigo Design Context

## Project
Multi-tenant e-commerce SaaS platform (Shopify competitor). Dashboard admin + visual page builder + storefront.

## Design Direction
- **Style**: Minimalist, Linear/Notion-inspired. Clean, professional, data-dense where needed.
- **Not**: Generic Bootstrap, purple gradients, cards-in-cards, template layouts.
- **Palette**: Neutral grays with semantic colors (success/warning/destructive/info). Primary: blue-600.
- **Typography**: System font stack. No Inter. Tight tracking on headings.
- **Density**: High for dashboard tables/lists, spacious for marketing/storefront pages.

## Design System Tokens
- Heights: h-7 (28px) panel controls, h-5 (20px) pills, h-8 (32px) toolbar
- Font sizes: 9px labels, 10px body, 11px inputs (editor); 13-14px dashboard body
- Spacing: gap-1 (4px), gap-2 (8px), gap-3 (12px)
- Radius: rounded-md controls, rounded-lg containers, rounded-full pills
- Opacity: /70 secondary text, /40 tertiary text. Never /20.
- Borders: border-sidebar-border for panels, border-border for content

## Tech Stack
- Next.js 16, React 19, Tailwind CSS v4, shadcn/ui
- Framer Motion for animations
- Supabase (Postgres + Auth + Storage)
- Drizzle ORM

## Anti-Patterns to Avoid
- Purple/violet gradients as primary palette
- Cards nested inside cards (cardocalypse)
- Inter font (use system stack)
- Generic hero sections with centered text + gradient background
- Placeholder "Lorem ipsum" content
- Low contrast text (below 4.5:1)
- Overuse of shadows (max 1 level of shadow per component)
